This HTML/JavaScript tool lets you visualize data from your Radiosonde Auto RX logs. The main.html file creates or updates a local IndexedDB in your browser, which stores the data for plotting.

ðŸ“Œ How to Use

1. Prepare Your Data

  • Copy the log files from your RadiosondeAutoRX folder into a separate directory.
  • Add a copy of the station.cfg file to the same directory.
Note: If you have more than ~2000 log files, process them in batches:
  • Copy a portion of the logs.
  • Open main.html and update the database.
  • Replace with the next batch and repeat until all logs are imported.

2. Update the Database

  • Open main.html in your web browser.
  • Click Update Database to parse and save the data.
  • You can add more logs later and update the DB again.

3. Generate and View Plots

  • Select the plot type from the menu.
  • Enter the required parameters.
  • View and analyze your data!
